Política de Privacidad – Corin

Última actualización: 24/02/2026

En Corin nos tomamos la privacidad en serio. Esta Política explica qué datos se recopilan, cómo se usan y cuáles son tus derechos al utilizar el servicio.

Al usar Corin, aceptás esta Política de Privacidad.

1. Qué datos recopila Corin

Corin recopila únicamente los datos necesarios para prestar el servicio. Esto puede incluir:

Datos de identificación y contacto

  • Nombre y apellido
  • Número de teléfono (WhatsApp)
  • CUIT
  • Información fiscal básica necesaria para facturación

Datos administrativos

  • Registro de sesiones (paciente y fecha)
  • Registro de pagos recibidos
  • Datos necesarios para emitir comprobantes electrónicos ante AFIP

Credenciales de ARCA/AFIP (para alta técnica)

Para habilitar la emisión de facturas en nombre del usuario, Corin puede solicitar que el usuario ingrese sus credenciales de ARCA/AFIP en un formulario seguro. Estas credenciales se envían al backend de Corin y se procesan de forma automatizada para realizar configuraciones/delegaciones en ARCA a través de AFIP SDK.

Corin no almacena estas credenciales ni están pensadas para ser vistas por ninguna persona del equipo: se usan de manera transitoria para completar el alta técnica.

Corin no solicita ni registra:

  • contenidos clínicos o terapéuticos
  • notas de sesiones
  • diagnósticos
  • información sensible relacionada con el tratamiento

El usuario es responsable de no enviar información clínica o sensible que no sea necesaria para fines administrativos.

2. Uso de la información

Los datos recopilados se utilizan exclusivamente para:

  • prestar y mejorar el funcionamiento de Corin
  • registrar información administrativa enviada por el usuario
  • asistir en la emisión de facturas electrónicas
  • realizar el alta técnica y las configuraciones/delegaciones necesarias en ARCA/AFIP para habilitar la emisión de facturas en nombre del usuario
  • realizar mediciones de uso y mejoras del producto
  • brindar soporte y comunicación relacionada con el servicio

Corin no vende, alquila ni comparte datos personales con fines comerciales.

3. Analytics y medición del servicio

Corin puede utilizar herramientas de analítica y medición de uso (por ejemplo, servicios como Amplitude) para:

  • entender cómo se utiliza el producto
  • detectar errores
  • mejorar la experiencia del usuario

Estos servicios procesan datos de uso de forma agregada y con fines exclusivamente estadísticos y de mejora del producto.

4. Servicios de terceros e infraestructura

Para funcionar, Corin utiliza servicios de terceros que proveen infraestructura técnica y operativa, los cuales pueden incluir, entre otros:

  • WhatsApp y proveedores de mensajería (por ejemplo, Twilio)
  • servicios de infraestructura y hosting (por ejemplo, Railway)
  • Web Services de AFIP
  • AFIP SDK (para automatizar configuraciones y delegaciones en ARCA/AFIP)
  • APIs y servicios técnicos necesarios para la facturación electrónica y operación del sistema

Estos proveedores procesan datos únicamente en la medida necesaria para cumplir su función técnica. Corin selecciona proveedores de uso habitual en el mercado y con estándares razonables de seguridad.

5. Credenciales y datos sensibles

Corin no almacena claves fiscales ni contraseñas sensibles del usuario.

Para habilitar la facturación, Corin puede procesar credenciales de ARCA/AFIP de forma transitoria (por ejemplo, para completar el alta técnica y configurar delegaciones). Esto se hace de manera automatizada a través de AFIP SDK. Corin no utiliza esas credenciales para otros fines.

6. Almacenamiento y seguridad

Corin aplica medidas razonables para proteger la información almacenada contra accesos no autorizados, pérdidas o usos indebidos.

Sin embargo, ningún sistema es completamente infalible. El usuario acepta que el uso del servicio implica riesgos inherentes a cualquier plataforma digital.

7. PDFs y comprobantes

Corin no almacena copias de los PDFs de las facturas emitidas.

Corin puede generar la representación en PDF de las facturas emitidas utilizando AFIP SDK y/o servicios de terceros necesarios para la operación del sistema. El usuario es responsable de descargar y conservar los comprobantes si así lo desea.

Los comprobantes pueden ser regenerados a partir de los datos fiscales almacenados (como numeración y CAE). Es responsabilidad del usuario descargar y conservar los comprobantes si así lo desea.

8. Conservación de los datos

Los datos se conservan mientras:

  • el usuario utilice el servicio, o
  • sean necesarios para cumplir con fines administrativos y fiscales

El usuario puede solicitar la eliminación de sus datos, salvo aquellos que deban conservarse por obligaciones legales.

9. Derechos del usuario

El usuario puede:

  • acceder a sus datos
  • solicitar correcciones
  • solicitar la eliminación de su información, en la medida legalmente posible

Para ejercer estos derechos, puede dejar de usar el servicio o solicitar asistencia por los canales disponibles.

10. Cambios en esta política

Corin puede actualizar esta Política de Privacidad. El uso continuado del servicio implica la aceptación de las versiones actualizadas.

11. Ley aplicable y jurisdicción

Esta Política se rige por las leyes de la República Argentina.

Cualquier controversia será sometida a los tribunales ordinarios de la Ciudad Autónoma de Buenos Aires, salvo disposición legal en contrario.

Si tenés dudas sobre esta Política de Privacidad, podés dejar de usar el servicio o consultar con un profesional de confianza.
